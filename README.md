## Prerequisites

- node 10.15.1
- npm 6.4.1
- angular-cli
- mongoose
- express

## Install

```sh
npm install
```

## Usage

#### To run backend
```sh
node app.js
```
#### To run frontend

```sh
cd client && ng serve
```
